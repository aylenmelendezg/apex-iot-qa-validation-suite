# Apex IoT QA Validation Suite

## Overview

A complete Quality Assurance validation project developed for an IoT Sensor Monitoring System.

This project simulates the validation process of a real-world sensor monitoring platform, including functional, validation, security, and performance testing.

The objective was to demonstrate the QA lifecycle from requirements analysis through final release approval.

---

## Project Scope

The system monitors environmental sensor data and provides:

- Temperature monitoring
- Humidity monitoring
- Alarm generation
- Dashboard visualization
- Historical data records
- Continuous 24-hour monitoring

---

## QA Artifacts Included

### Requirements Traceability Matrix (RTM)

Mapping between requirements and test cases to ensure complete coverage.

### Test Cases

15 test cases designed and executed covering:

- Functional Testing
- Validation Testing
- Security Testing
- Performance Testing

### Test Execution Report

Execution evidence and results for all test scenarios.

### Final Validation Report

Final assessment and release recommendation.
